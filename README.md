# 🧮 Kalkulator

**Kalkulator** to aplikacja stworzona w technologii .NET MAUI, umożliwiająca wykonywanie podstawowych operacji arytmetycznych.

## 🛠️ Technologie

- ⚙️ .NET MAUI
- 🖥️ C#

## 🚀 Funkcjonalności

✅ Dodawanie<br>
✅ Odejmowanie<br>
✅ Mnożenie<br>
✅ Dzielenie<br>
✅ Cofanie pojedyńczej cyfry lub całej liczby<br>
